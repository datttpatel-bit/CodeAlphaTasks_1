# Hangman Game 🎮

A simple command-line Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship.

## Description

The Hangman Game is a word-guessing game where the player tries to guess a hidden word one letter at a time. The player has a limited number of incorrect attempts before the game ends.

## Features

- Random word selection
- User-friendly command-line interface
- Tracks correct and incorrect guesses
- Limited attempts for challenging gameplay
- Win and lose conditions

## Technologies Used

- Python 3
- Random Module
- Loops and Conditional Statements
